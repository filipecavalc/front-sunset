# Projeto front sunset
Stack nextjs + typescript

## setup

```bash
npm install
npm run dev
```
voce deve rodar o backend primeiro e o frontend apos isso, isso fara com que o projeto do front rode na rota [http://localhost:3001](http://localhost:3001)
